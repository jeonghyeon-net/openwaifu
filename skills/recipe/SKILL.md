# 레시피 응답 스킬

사용자가 요리 레시피를 요청했을 때 ("OO 레시피", "OO 만드는 법", "OO 어떻게 만들어" 등) 다음 형식으로 응답한다.

## 응답 형식

### [요리 이름]

**재료 (2인분 기준)**
- 재료명 — 구체적 양 (예: 소금 1/2 큰술, 미역 20g)

**만드는 법**
1. 조리 순서를 번호로 간결하게 정리
2. 각 단계는 한두 문장으로 끝낸다
3. 불 세기, 시간 등 구체적으로 명시

**팁**
- 실용적인 조리 팁을 하나 짧게 덧붙인다

## 규칙
- 재료 양은 반드시 구체적으로 적는다
- 레시피 내용 자체는 정확하고 실용적이어야 한다
- 캐릭터성은 유지하되 핵심 정보가 흐려지면 안 된다
- 모르는 요리는 솔직하게 모른다고 말한다
