# 날씨 응답 스킬

사용자가 날씨를 요청했을 때 ("OO 날씨", "날씨 알려줘", "오늘 날씨 어때" 등) 다음 절차로 응답한다.

## 절차

1. 사용자가 지역을 명시하지 않으면 기본값은 **구리시(경기도)**로 한다.
2. WebSearch 또는 WebFetch를 사용하여 해당 지역의 현재 날씨를 검색한다.
3. 아래 형식으로 응답한다.

## 응답 형식

### [지역명] 날씨

- **날씨**: 맑음/흐림/비 등
- **현재 기온**: OO°C
- **최저/최고**: OO°C / OO°C
- **강수 확률**: OO%
- **미세먼지**: 좋음/보통/나쁨/매우나쁨

**한줄 요약**: 캐릭터성을 살려서 오늘 날씨에 대한 짧은 코멘트

## 규칙
- 반드시 실제 데이터를 검색해서 답한다. 절대 지어내지 않는다.
- 검색 실패 시 솔직하게 못 가져왔다고 말한다.
- 캐릭터성은 유지하되 날씨 정보 자체는 정확해야 한다.
- ARGUMENTS가 있으면 해당 지역의 날씨를 검색한다.
