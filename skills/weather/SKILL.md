# 날씨 응답 스킬

사용자가 날씨를 요청했을 때 ("OO 날씨", "날씨 알려줘", "오늘 날씨 어때" 등) 다음 절차로 응답한다.

## 절차

1. 사용자가 지역을 명시하지 않으면 기본값은 **구리시(경기도)**로 한다.
2. WebSearch를 사용하여 해당 지역의 현재 날씨를 검색한다.
3. 아래 형식으로 응답한다.

## 응답 형식

**[지역명] 오늘의 날씨**

> 날씨상태이모지 **현재 기온** OO°C (체감 OO°C)
> 최저 OO°C / 최고 OO°C
> 강수확률 OO% | 습도 OO%
> 미세먼지 좋음/보통/나쁨/매우나쁨

---

**오늘 뭐 입지?**
날씨와 기온에 맞는 구체적인 패션 코디를 1~2줄로 추천한다. 아우터, 상의, 하의, 신발까지 포함. 비 오면 우산/방수 아이템도 언급.

---

캐릭터성을 살린 한줄 코멘트로 마무리.

## 날씨 이모지 기준
- 맑음: ☀️
- 구름 많음: ⛅
- 흐림: ☁️
- 비: 🌧️
- 눈: ❄️
- 뇌우: ⛈️

## 규칙
- 반드시 실제 데이터를 검색해서 답한다. 절대 지어내지 않는다.
- 검색 실패 시 솔직하게 못 가져왔다고 말한다.
- 캐릭터성은 유지하되 날씨 정보 자체는 정확해야 한다.
- 패션 추천은 기온과 날씨 상태에 맞게 실용적으로 한다.
- ARGUMENTS가 있으면 해당 지역의 날씨를 검색한다.
